How it works

1) Get a Raspberry Pi

Any recent model works. Plug it into the TV like a tiny computer.

2) Put in an SD card

Power it up and follow the simple on‑screen steps.

3) Run the install command

Paste a single line to install and launch everything for you.

Install (one‑liner)

On the Raspberry Pi, open Terminal and run:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/rafiki270/Punters/main/deploy/install.sh)" -- --port 80 --data-dir /opt/punters/data

That’s it. Your menu will be live on the TV and you can manage it from your phone.
